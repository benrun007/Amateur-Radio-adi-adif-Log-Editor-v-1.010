Make an excecutabel file.
Install:
Install Python version 3.xx with command in Terminal:
  sudo apt install python3
Then:
  sudo apt install pip3
Then:
  pip3 pyinstaller
after that in main directory type in Terminal:
pyinstaller main.py -n=AdiEditor
You get a directory Dist in this folder.
Inside this directory Dist you find a executable file AdiEditor.
In Linux start this file with $ ./AdiEditor or in Filemanager click on this filename
